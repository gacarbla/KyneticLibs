# Kynetic Web Libraries Monorepo

This monorepo contains various packages developed by Kynetic Web. Each package is maintained in its own directory under the `packages` folder.

## Packages

- `@kyneticweb/console`: A console management library for Node.js applications.
- `@kyneticweb/container`: A dependency injection container for managing application dependencies.
- `@kyneticweb/counter`: A simple counter utility for tracking counts and statistics.
- `@kyneticweb/parser`: A parsing library for various data formats.
- `@kyneticweb/versions`: A versioning library to manage application versions.

## Installation

```bash
npm install @kyneticweb/libs
```
